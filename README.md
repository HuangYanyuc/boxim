
### 项目介绍

仿微信实现的网页版聊天软件，后端使用springboot+netty实现，前端使用vue。支持私聊、群聊、离线消息、发送图片、文件、好友在线状态显示等功能，服务器支持集群化部署。



#### 项目结构
|  模块  |     功能 |
|-------------|------------|
| im-platform | 与页面进行交互，处理业务请求 |
| im-server   | 推送聊天消息|
| im-common   |公共包
