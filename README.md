
### 项目介绍

仿微信实现的网页版聊天软件，后端使用springboot+netty实现，前端使用vue。
目前是1.0.0版本，仅支持私聊文字消息，后续会陆续上线群聊、发送表情、图片等功能

### 主要功能

目前功能：

- 好友在线状态显示
- 实时发送私聊消息，上线后自动接收离线消息
- 支持服务器集群化部署


#### 项目结构
|  模块  |     功能 |
|-------------|------------|
| im-platform | 与页面进行交互，处理业务请求 |
| im-server   | 推送聊天消息       |
| im-common    | 公共包
